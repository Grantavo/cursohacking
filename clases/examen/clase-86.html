1. ¿Cuál de los siguientes comandos se usa para listar archivos en un directorio?

  • A) pwd
  • B) ls
  • C) cd
  • D) mkdir

2. ¿Qué permiso representa el número 755 en octal?

  • A) rwxr-xr-x
  • B) rwxrwxrwx
  • C) rwxr--r--
  • D) r-xr-xr-x

3. ¿Qué hace el comando chmod +x script.sh?

  • A) Cambia el propietario del archivo
  • B) Añade permiso de ejecución al archivo
  • C) Elimina el permiso de escritura
  • D) Cambia el grupo del archivo

4. ¿Cuál de los siguientes operadores redirige la salida estándar a un archivo SOBREESCRIBIÉNDOLO?

  • A) >
  • B) >>
  • C) 2>
  • D) <

5. ¿Qué hace el comando grep -r "root" /etc?

  • A) Busca la palabra "root" solo en archivos del directorio actual
  • B) Busca recursivamente la palabra "root" en todos los archivos dentro de /etc
  • C) Reemplaza "root" por otra palabra en /etc
  • D) Elimina todas las líneas que contengan "root"

6. ¿Cuál es la función del bit SUID en un archivo?

  • A) Permite que cualquier usuario pueda escribir en el archivo
  • B) Hace que el archivo se ejecute con los permisos del propietario
  • C) Impide que el archivo sea eliminado
  • D) Borra el archivo automáticamente al cerrarse

7. ¿Qué tubería (pipe) usarías para contar cuántos archivos hay en un directorio?

  • A) ls | wc -l
  • B) ls > wc -l
  • C) ls && wc -l
  • D) ls; wc -l

8. En Bash, ¿qué estructura usarías para repetir un bloque de código mientras se cumpla una condición?

  • A) if
  • B) for
  • C) while
  • D) case

9. ¿Cuál es el comando correcto para conectarse por SSH a un servidor con IP 192.168.1.100 y usuario admin?

  • A) ssh 192.168.1.100 admin
  • B) ssh admin@192.168.1.100
  • C) ssh admin:192.168.1.100
  • D) ssh -u admin 192.168.1.100

10. ¿Qué hace el comando netcat -lvnp 4444?

  • A) Conecta a un servidor remoto en el puerto 4444
  • B) Escucha conexiones entrantes en el puerto 4444
  • C) Escanea los puertos de una IP remota
  • D) Transfiere un archivo a través del puerto 4444

11. ¿Qué expresión regular usarías con grep para buscar líneas que empiecen por "error"?

  • A) grep "error$" archivo
  • B) grep "^error" archivo
  • C) grep "error" archivo
  • D) grep "error*" archivo

12. ¿Cuál de los siguientes comandos de sed reemplaza la primera ocurrencia de "foo" por "bar" en cada línea?

  • A) sed 's/foo/bar/g' archivo
  • B) sed 's/foo/bar/' archivo
  • C) sed 's/foo/bar/2' archivo
  • D) sed 's/foo/bar/i' archivo

13. ¿Qué comando de awk imprime la segunda columna de un archivo?

  • A) awk '{print $1}' archivo
  • B) awk '{print $2}' archivo
  • C) awk '{print $0}' archivo
  • D) awk '{print $NF}' archivo

14. ¿Qué hace echo "Linux" | base64?

  • A) Descodifica la cadena "Linux" de base64
  • B) Codifica la cadena "Linux" a base64
  • C) Comprime la cadena "Linux"
  • D) Calcula el hash MD5 de "Linux"

  • A) git commit
  • B) git push
  • C) git pull
  • D) git fetch

  • A) No hay diferencia, hacen lo mismo
  • B) git pull descarga cambios y los fusiona; git fetch solo descarga
