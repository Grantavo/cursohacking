Preparando Ubuntu para el curso

Duración: — · Sección 1: Introducción

Ya tienes Ubuntu instalado y sabemos por qué es una gran elección. Ahora vamos a prepararlo con todo lo necesario para el curso. Sigue estos pasos en tu terminal y tendrás un entorno listo para aprender.

1. Actualizar el sistema

Lo primero es asegurarte de que tu Ubuntu está al día. Abre una terminal y ejecuta:

sudo apt update && sudo apt upgrade -y

apt update refresca la lista de paquetes disponibles y apt upgrade instala las actualizaciones. Dependiendo de cuándo actualizases por última vez, puede tardar unos minutos.

2. Habilitar repositorios Universe y Multiverse

Ubuntu incluye varios repositorios. Los que nos interesan son:

  • Main — paquetes oficiales mantenidos por Canonical (ya activo).
  • Universe — paquetes mantenidos por la comunidad (muchas herramientas útiles).
  • Multiverse — paquetes con licencias no libres (códecs, drivers, etc.).

Actívalos con:

sudo add-apt-repository universe
sudo add-apt-repository multiverse
sudo apt update

3. Instalar herramientas esenciales

Vamos a instalar las herramientas que usaremos a lo largo del curso. Ejecuta este comando completo:

sudo apt install -y curl wget git vim netcat-openbsd nmap openssh-server tmux htop tree

Te explico cada una:

  • curl / wget — descargar archivos y hacer peticiones web desde terminal.
  • git — control de versiones y clonar repositorios.
  • vim — editor de texto potente para la terminal.
  • netcat-openbsd — conexiones TCP/UDP, escaneo de puertos, transferencia de archivos.
  • nmap — escaneo de redes y puertos.
  • openssh-server — servidor SSH para conexiones remotas.
  • tmux — multiplexor de terminal (varias sesiones en una ventana).
  • htop — monitor de procesos interactivo y más visual que top.
  • tree — muestra la estructura de directorios en árbol.
📦 Más adelante instalaremos otras herramientas según las necesitemos (como hydra, john, metasploit, etc.). Por ahora con estas es suficiente para empezar.

4. Configurar el firewall (opcional)

Ubuntu incluye ufw (Uncomplicated Firewall). Si quieres mantenerlo activo mientras aprendes, permite SSH si lo necesitas:

sudo ufw allow OpenSSH
sudo ufw enable
sudo ufw status

5. Crear estructura de directorios

Te recomiendo crear una carpeta para organizar los proyectos del curso:

mkdir -p ~/cursos/linux/{scripts,proyectos,notas,recursos}

Esto crea una estructura como:

~/cursos/linux/
├── scripts/     → Aquí irán tus scripts en Bash
├── proyectos/   → Proyectos del curso
├── notas/       → Tus apuntes y resúmenes
└── recursos/    → Archivos de apoyo

6. Hacer una copia de seguridad

Antes de empezar a hacer cambios en el sistema, es recomendable tener un respaldo por si algo sale mal:

  • Timeshift — herramienta similar a los snapshots de VirtualBox pero para tu sistema real. Instálala con sudo apt install timeshift y crea un snapshot inicial.
  • Si no quieres instalar nada adicional, al menos asegúrate de tener tus archivos importantes respaldados.
⚠️ Nota importante: Con 4 GB de RAM tienes suficiente para todo el curso. Ubuntu con GNOME consume ~1-1.5 GB en reposo, y las herramientas que usaremos son ligeras. Si notas el sistema lento, puedes cerrar navegadores y otras aplicaciones mientras practicas.

Con todo esto listo, ya tenemos Ubuntu preparado. En la siguiente clase daremos los primeros pasos en el entorno.
