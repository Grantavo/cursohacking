Primeros pasos en Ubuntu

Duración: — · Sección 1: Introducción

Ya tienes Ubuntu instalado y preparado. Ahora vamos a familiarizarnos con el entorno y la terminal. Esta clase sienta las bases para todo lo que viene después.

El escritorio de Ubuntu

Ubuntu viene por defecto con el escritorio GNOME. Es un entorno moderno, limpio y muy intuitivo. Tómate un momento para explorarlo: el dock a la izquierda, el menú de actividades en la esquina superior izquierda y el panel superior con la hora y el calendario.

Si usas otra variante de Ubuntu (Kubuntu con KDE, Xubuntu con Xfce, etc.), no te preocupes: la terminal y los comandos son los mismos. El escritorio es solo la capa visual.

Abrir la terminal

La terminal es tu herramienta principal. Puedes abrirla de varias formas:

  • Atajo de teclado: Ctrl + Alt + T (el más rápido).
  • Menú de actividades: Super (tecla Windows) → escribe "terminal" → Enter.
  • Clic derecho: En algunas versiones, "Abrir terminal" desde el escritorio.

Personalizar la terminal

Haz que la terminal sea tuya. Ve a Editar → Preferencias (o el menú de hamburguesa) y juega con:

  • Perfil: Puedes crear un perfil nuevo llamado "Curso Linux" para tener una configuración separada.
  • Colores: Elige un esquema oscuro (menos fatiga visual). Puedes usar temas como Dracula o Solarized Dark.
  • Fuente: Cambia a JetBrains Mono o Fira Code (más legible para código). Tamaño 12-14.
  • Transparencia: Un fondo ligeramente translúcido queda profesional y permite ver ventanas detrás.

El concepto de sudo

En Ubuntu no trabajamos como root directamente. A diferencia de Kali Linux, donde inicias sesión como root por defecto, Ubuntu sigue un modelo más seguro: usas un usuario normal y cuando necesitas permisos elevados usas sudo.

Acción En Ubuntu En Kali
Instalar un paquete sudo apt install nano apt install nano
Actualizar el sistema sudo apt update apt update
Editar un archivo de sistema sudo vim /etc/hosts vim /etc/hosts
Iniciar sesión como root sudo -i (requiere contraseña de usuario) Ya eres root directamente
💡 Importante: Todos los comandos que aprenderemos en el curso funcionan exactamente igual en Ubuntu que en Kali. La única diferencia es que en Ubuntu necesitarás anteponer sudo para operaciones privilegiadas. Esto es en realidad más seguro y es lo que se hace en entornos profesionales.

Crear un usuario de prácticas (opcional)

Si quieres tener un usuario específico para el curso, puedes crearlo:

sudo adduser alumnolinux

Te pedirá contraseña y algunos datos. Luego puedes cambiar a ese usuario con:

su - alumnolinux

Configurar alias útiles en bashrc

El archivo ~/.bashrc se ejecuta cada vez que abres una terminal. Vamos a añadir algunos alias que te harán la vida más fácil. Ábrelo con:

vim ~/.bashrc

Añade estas líneas al final:

alias ll='ls -la'
alias la='ls -A'
alias cls='clear'
alias update='sudo apt update && sudo apt upgrade'

Para que los cambios surtan efecto sin cerrar la terminal, ejecuta:

source ~/.bashrc
  • ll — lista archivos con detalles y ocultos.
  • la — lista archivos incluyendo los que empiezan con punto.
  • cls — limpia la pantalla (como clear pero más rápido de escribir).
  • update — actualiza el sistema de un solo golpe.

¿Y ahora qué?

Con Ubuntu preparado, la terminal personalizada y los alias configurados, estás listo para empezar con los comandos. Recuerda: todo lo que aprenderás desde la siguiente clase funciona exactamente igual en Ubuntu. Los conceptos de Linux son universales.

⚠️ Diferencia clave: Si vienes de Kali o ves tutoriales donde el usuario es root, recuerda que en Ubuntu debes usar sudo. Si un comando falla con "Permission denied", probablemente solo necesites añadir sudo al principio.

¡Nos vemos en la siguiente sección, donde empezaremos con los comandos básicos de Linux!
