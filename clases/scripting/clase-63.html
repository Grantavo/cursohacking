Introducción al Proyecto 1 — Buscador de máquinas HTB

Sección 6: Scripting en Bash — Principiante a Avanzado

Bienvenido al primer proyecto práctico de la sección de scripting. Vamos a construir un buscador de máquinas de HackTheBox completamente funcional usando Bash. Este proyecto integra todo lo que hemos aprendido: variables, funciones, arrays, bucles, y añade conceptos nuevos como peticiones HTTP con curl y procesamiento JSON con jq.

¿Qué vamos a construir?

Un script de terminal que:

  • Se conecte a la API pública de HackTheBox
  • Busque máquinas por nombre, IP, sistema operativo o dificultad
  • Muestre los resultados en una tabla formateada con colores
  • Permita paginación, caché y modo interactivo

Requisitos

📦 Paquetes necesarios:
  • curl — para hacer peticiones HTTP a la API
  • jq — para procesar JSON desde la terminal
  • tput — para colores y formato en la terminal

Verifica que están instalados:

which curl jq tput
# Si falta alguno:
sudo apt install -y curl jq

Conceptos nuevos

curl

Herramienta para transferir datos con URLs. La usaremos para consultar la API de HTB:

curl -s "https://www.hackthebox.com/api/machines/list" \
  -H "Authorization: Bearer TU_TOKEN"

jq

Procesador JSON de línea de comandos. Permite filtrar y transformar JSON:

curl -s "https://api.example.com/data" | jq '.results[] | {name: .name, os: .os}'

API de HackTheBox

HTB expone una API REST que devuelve información de máquinas en formato JSON. Necesitarás un token de API desde tu panel de usuario.

⚠️ Importante: No compartas tu token de API. Lo manejaremos mediante una variable de entorno HTB_TOKEN para no hardcodearlo.

Estructura del script

Planificamos el script en estas secciones:

# === FUNCIONES ===
# === ANÁLISIS DE ARGUMENTOS ===
# === LÓGICA PRINCIPAL ===
