Scripting en Bash [1-6]

Duración: 23:52 · Proyecto 1: Buscador HTB

Empezamos a escribir el script del buscador de HTB. En esta clase configuraremos la base: shebang, variables de configuración, colores con tput, la función print_banner y el análisis de argumentos con getopts.

Configuración inicial

Creamos el archivo htb_search.sh con el shebang y las variables de configuración:

#!/bin/bash

# === CONFIGURACIÓN ===
HTB_TOKEN="${HTB_TOKEN:-}"
API_URL="https://www.hackthebox.com/api/machines"
CACHE_DIR="${HOME}/.cache/htb_search"
CACHE_TTL=300  # 5 minutos

# Colores con tput
BOLD=$(tput bold)
RED=$(tput setaf 1)
GREEN=$(tput setaf 2)
YELLOW=$(tput setaf 3)
BLUE=$(tput setaf 4)
MAGENTA=$(tput setaf 5)
CYAN=$(tput setaf 6)
RESET=$(tput sgr0)

Función print_banner

Mostramos un banner ASCII al iniciar el script:

print_banner() {

Análisis de argumentos con getopts

Procesamos las opciones de línea de comandos:

# === ANÁLISIS DE ARGUMENTOS ===
MODE="search"
OS=""
DIFFICULTY=""
NAME=""
INTERACTIVE=false
DEBUG=false

usage() {
  echo "Uso: $0 [opciones]"
  echo "  -n     Buscar por nombre"
  echo "  -o         Filtrar por SO (Linux/Windows)"
  echo "  -d  Filtrar por dificultad (easy/medium/hard/insane)"
  echo "  -i             Modo interactivo"
  echo "  -D             Modo debug"
  echo "  -h             Mostrar ayuda"
  exit 0
}

while getopts "n:o:d:iDh" opt; do
  case $opt in
    n) NAME="$OPTARG" ;;
    o) OS="$OPTARG" ;;
    d) DIFFICULTY="$OPTARG" ;;
    i) INTERACTIVE=true ;;
    D) DEBUG=true ;;
    h) usage ;;
    *) usage ;;
  esac
done

Validación del token

🔐 Token de API: Establece tu token como variable de entorno antes de ejecutar:
export HTB_TOKEN="tu_token_aqui"
O créalo en un archivo .env y cárgalo con source .env. 
# Validar token
if [[ -z "$HTB_TOKEN" ]]; then
  echo "${RED}${BOLD}Error:${RESET} HTB_TOKEN no está configurado."
  echo "Ejecuta: export HTB_TOKEN='tu_token'"
  exit 1
fi
