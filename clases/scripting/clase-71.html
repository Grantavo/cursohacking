Scripting en Bash [1-15]

Duración: 8:24 · Proyecto 2: Ruleta de casino

Damos los primeros pasos con la ruleta. Creamos el script con las variables globales (saldo, apuesta, número elegido), la función de bienvenida y la entrada del usuario.

Función de bienvenida

Mostramos un mensaje de bienvenida con las instrucciones básicas:

bienvenida() {
  clear
  echo "${BOLD}${AMARILLO}"
  echo '  ╔══════════════════════════════════╗'
  echo '  ║     🎰 CASINO ROULETTE v1.0      ║'
  echo '  ║     Bash Edition — Terminal       ║'
  echo '  ╚══════════════════════════════════╝'
  echo "${RESET}"
  echo "${CIAN}Bienvenido a la ruleta de casino en Bash${RESET}"
  echo "Saldo inicial: ${VERDE}\$${SALDO}${RESET}"
  echo
  echo "${AMARILLO}Reglas:${RESET}"
  echo "  • Elige un tipo de apuesta y una cantidad"
  echo "  • La ruleta genera un número aleatorio (0-36)"
  echo "  • Si aciertas, ganas según la tabla de pagos"
  echo "  • Si pierdes, pierdes tu apuesta"
  echo "  • ¡No apuestes lo que no puedas perder!"
  echo
}

Entrada del usuario

Solicitamos nombre y saldo inicial (con valor por defecto):

iniciar_juego() {
  bienvenida

  # Nombre del jugador
  read -r -p "Introduce tu nombre: " NOMBRE
  NOMBRE="${NOMBRE:-Jugador}"

  # Saldo inicial personalizado
  read -r -p "Saldo inicial [1000]: " saldo_in
  SALDO="${saldo_in:-1000}"

  # Validar que el saldo sea un número positivo
  if ! [[ "$SALDO" =~ ^[0-9]+$ ]] || [[ "$SALDO" -eq 0 ]]; then
    print_error "El saldo debe ser un número positivo"
    SALDO=1000
  fi

  print_success "¡Bienvenido, ${NOMBRE}! Saldo: \$${SALDO}"
  sleep 1
}
💡 Expansión de variables con valores por defecto: NOMBRE="${NOMBRE:-Jugador}" asigna "Jugador" si la variable está vacía. Es una forma limpia de establecer defaults en Bash.
