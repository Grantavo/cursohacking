Scripting en Bash [3-15] (GRATIS)

Duración: 17:09 · Proyecto 2: Ruleta de casino

Implementamos la lógica completa de apuestas: pleno (1 número), docena, mitad, color y par/impar. Cada tipo tiene su propio cálculo de pagos según las reglas de la ruleta europea.

Tipos de apuesta

Definimos constantes para los tipos de apuesta:

declare -A PAGOS
PAGOS[pleno]=35
PAGOS[docena]=2
PAGOS[mitad]=1
PAGOS[color]=1
PAGOS[par_impar]=1

declare -A DESCRIPCIONES
DESCRIPCIONES[pleno]="Un número específico (35:1)"
DESCRIPCIONES[docena]="1-12, 13-24, 25-36 (2:1)"
DESCRIPCIONES[mitad]="1-18 (baja) o 19-36 (alta) (1:1)"
DESCRIPCIONES[color]="Rojo o negro (1:1)"
DESCRIPCIONES[par_impar]="Par o impar (1:1)"

Lógica de apuesta: pleno

apostar_pleno() {
  local numero_apostado=$1
  local cantidad=$2
  local resultado_numero=$3

  if [[ "$numero_apostado" -eq "$resultado_numero" ]]; then
    local ganancia=$(( cantidad * PAGOS[pleno] ))
    echo "$ganancia"
  else
    echo 0
  fi
}

Lógica de apuesta: docena

apostar_docena() {
  local docena=$1    # 1, 2 o 3
  local cantidad=$2
  local resultado=$3

  local min max
  case $docena in
    1) min=1;  max=12 ;;
    2) min=13; max=24 ;;
    3) min=25; max=36 ;;
    *) echo 0; return ;;
  esac

  if (( resultado >= min && resultado <= max )); then
    local ganancia=$(( cantidad * PAGOS[docena] ))
    echo "$ganancia"
  else
    echo 0
  fi
}

Lógica de apuesta: mitad (baja/alta)

apostar_mitad() {
  local eleccion=$1    # "baja" (1-18) o "alta" (19-36)
  local cantidad=$2
  local resultado=$3

  if [[ "$resultado" -eq 0 ]]; then
    echo 0
    return
  fi

  if [[ "$eleccion" == "baja" ]] && (( resultado <= 18 )); then
    echo $(( cantidad * PAGOS[mitad] ))
  elif [[ "$eleccion" == "alta" ]] && (( resultado >= 19 )); then
    echo $(( cantidad * PAGOS[mitad] ))
  else
    echo 0
  fi
}

Lógica de apuesta: color

apostar_color() {
  local color_elegido=$1  # "rojo" o "negro"
  local cantidad=$2
  local resultado=$3
  local color_resultado
  color_resultado=$(determinar_color "$resultado")

  if [[ "$color_resultado" == "verde" ]]; then
    echo 0
    return
  fi

  if [[ "$color_elegido" == "$color_resultado" ]]; then
    echo $(( cantidad * PAGOS[color] ))
  else
    echo 0
  fi
}

Lógica de apuesta: par/impar

apostar_par_impar() {
  local eleccion=$1    # "par" o "impar"
  local cantidad=$2
  local resultado=$3
  local paridad
  paridad=$(determinar_paridad "$resultado")

  if [[ "$paridad" == "ninguno" ]]; then
    echo 0
    return
  fi

  if [[ "$eleccion" == "$paridad" ]]; then
    echo $(( cantidad * PAGOS[par_impar] ))
  else
    echo 0
  fi
}

Función unificada de apuesta

procesar_apuesta() {
  local tipo=$1
  local eleccion=$2
  local cantidad=$3
  local resultado=$4

  case $tipo in
    pleno)     apostar_pleno "$eleccion" "$cantidad" "$resultado" ;;
    docena)    apostar_docena "$eleccion" "$cantidad" "$resultado" ;;
    mitad)     apostar_mitad "$eleccion" "$cantidad" "$resultado" ;;
    color)     apostar_color "$eleccion" "$cantidad" "$resultado" ;;
    par_impar) apostar_par_impar "$eleccion" "$cantidad" "$resultado" ;;
    *)         echo 0 ;;
  esac
}
📊 Cálculo de pagos: En la ruleta europea, la ventaja de la casa es del 2.7% (el 0 verde). Esto significa que estadísticamente, por cada $100 apostados, recuperarás ~$97.30. ¡La casa siempre gana a largo plazo!
