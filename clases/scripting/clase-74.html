Scripting en Bash [4-15]

Duración: 18:33 · Proyecto 2: Ruleta de casino

Creamos el bucle principal del juego con un menú interactivo, validación de entrada (números, rangos) y control de flujo con case.

Menú interactivo

Usamos select para presentar las opciones al jugador:

mostrar_menu() {
  echo
  echo "${BOLD}${NOMBRE} — Saldo: ${VERDE}\$${SALDO}${RESET}"
  echo
  select opt in "🎯 Apostar" "📊 Estadísticas" "💾 Guardar partida" "🔧 Debug" "🚪 Salir"; do
    case $opt in
      "🎯 Apostar")
        realizar_apuesta
        break
        ;;
      "📊 Estadísticas")
        mostrar_estadisticas
        break
        ;;
      "💾 Guardar partida")
        guardar_partida
        break
        ;;
      "🔧 Debug")
        menu_debug
        break
        ;;
      "🚪 Salir")
        echo "${AMARILLO}¡Gracias por jugar, ${NOMBRE}!${RESET}"
        exit 0
        ;;
      *)
        echo "${ROJO}Opción inválida${RESET}"
        ;;
    esac
  done
}

Validación de entrada

Verificamos que el usuario introduzca valores válidos:

validar_cantidad() {
  local cantidad=$1

  # Debe ser un número
  if ! [[ "$cantidad" =~ ^[0-9]+$ ]]; then
    print_error "Debes introducir un número válido"
    return 1
  fi

  # Debe ser positiva
  if [[ "$cantidad" -le 0 ]]; then
    print_error "La cantidad debe ser mayor que 0"
    return 1
  fi

  # No puede exceder el saldo
  if [[ "$cantidad" -gt "$SALDO" ]]; then
    print_error "No tienes suficiente saldo (Saldo: \$${SALDO})"
    return 1
  fi

  return 0
}

validar_numero_ruleta() {
  local num=$1

  if ! [[ "$num" =~ ^[0-9]+$ ]]; then
    print_error "Debes introducir un número"
    return 1
  fi

  if [[ "$num" -lt 0 ]] || [[ "$num" -gt 36 ]]; then
    print_error "El número debe estar entre 0 y 36"
    return 1
  fi

  return 0
}

Flujo de apostar

Coordinamos la entrada del usuario con la lógica de apuesta:

realizar_apuesta() {
  # 1. Elegir tipo de apuesta
  echo "${BOLD}Tipo de apuesta:${RESET}"
  select tipo in "Pleno" "Docena" "Mitad" "Color" "Par/Impar" "Cancelar"; do
    case $tipo in
      "Pleno")
        read -r -p "Número (0-36): " NUMERO_ELEGIDO
        validar_numero_ruleta "$NUMERO_ELEGIDO" || continue
        TIPO_APUESTA="pleno"
        break
        ;;
      "Docena")
        select docena in "1-12" "13-24" "25-36" "Cancelar"; do
          [[ -z "$docena" ]] && continue
          case $docena in
            "1-12")   ELECCION=1 ;;
            "13-24")  ELECCION=2 ;;
            "25-36")  ELECCION=3 ;;
            "Cancelar") return ;;
          esac
          TIPO_APUESTA="docena"
          break
        done
        break
        ;;
      "Mitad")
        select ELECCION in "Baja (1-18)" "Alta (19-36)" "Cancelar"; do
          [[ -z "$ELECCION" ]] && continue
          [[ "$ELECCION" == "Cancelar" ]] && return
          TIPO_APUESTA="mitad"
          break
        done
        break
        ;;
      "Color")
        select ELECCION in "Rojo" "Negro" "Cancelar"; do
          [[ -z "$ELECCION" ]] && continue
          [[ "$ELECCION" == "Cancelar" ]] && return
          ELECCION=$(echo "$ELECCION" | tr '[:upper:]' '[:lower:]')
          TIPO_APUESTA="color"
          break
        done
        break
        ;;
      "Par/Impar")
        select ELECCION in "Par" "Impar" "Cancelar"; do
          [[ -z "$ELECCION" ]] && continue
          [[ "$ELECCION" == "Cancelar" ]] && return
          ELECCION=$(echo "$ELECCION" | tr '[:upper:]' '[:lower:]')
          TIPO_APUESTA="par_impar"
          break
        done
        break
        ;;
      "Cancelar") return ;;
    esac
    break
  done

  # 2. Cantidad a apostar
  while true; do
    read -r -p "Cantidad a apostar: " APUESTA
    validar_cantidad "$APUESTA" && break
  done

  # 3. Girar ruleta y procesar
  ejecutar_turno
}

Control de flujo con case

El esqueleto del juego usa case para manejar todas las acciones:

ejecutar_turno() {
  # Deducir apuesta del saldo
  (( SALDO -= APUESTA ))

  # Girar la ruleta
  local resultado
  resultado=$(girar_ruleta)
  IFS='|' read -r numero color paridad <<< "$resultado"

  # Calcular ganancia
  local ganancia
  ganancia=$(procesar_apuesta "$TIPO_APUESTA" "$ELECCION" "$APUESTA" "$numero")

  if [[ "$ganancia" -gt 0 ]]; then
    (( SALDO += ganancia ))
    local beneficio=$(( ganancia - APUESTA ))
    echo "${VERDE}${BOLD}¡Has ganado!${RESET} +${beneficio}"
  else
    echo "${ROJO}${BOLD}Has perdido${RESET} -${APUESTA}"
  fi

  # Guardar en historial
  HISTORIAL+=("$numero:$color:$paridad:$ganancia")
  (( TOTAL_JUGADAS++ ))

  # Comprobar game over
  if [[ "$SALDO" -le 0 ]]; then
    game_over
  fi
}
⚠️ Validación defensiva: Siempre valida la entrada del usuario antes de procesarla. Nunca confíes en que el usuario introducirá datos correctos. Esto evita bugs y comportamientos inesperados.
