Lectura e interpretación de permisos [2-2]

Duración: 3:43 · Sección 2: Temario

Ahora que sabes leer los permisos, toca aprender a cambiarlos. El comando chmod en modo simbólico te permite modificar permisos de forma intuitiva.

Modo simbólico: estructura

La sintaxis es: chmod [categoría][operador][permiso] archivo

CategoríaOperadorPermiso
u (user)+ (añadir)r (read)
g (group)- (quitar)w (write)
o (others)= (asignar exacto)x (execute)
a (all)

Ejemplos prácticos

chmod u+x script.sh          # Añade ejecución al dueño
chmod g-w archivo.txt        # Quita escritura al grupo
chmod o+r documento.pdf      # Añade lectura a otros
chmod a+x programa           # Añade ejecución a todos (dueño, grupo, otros)
chmod u=rw,go=r archivo      # Dueño: rw, grupo y otros: solo r
💡 Puedes combinar múltiples cambios separándolos con comas: chmod u+w,go-x archivo añade escritura al dueño y quita ejecución a grupo y otros.

Progresión de ejemplos

1 Crear un script y hacerlo ejecutable 
echo 'echo "Hola"' > saludo.sh
chmod u+x saludo.sh
./saludo.sh                  # Ahora funciona
2 Proteger un archivo contra modificaciones 
chmod u-w notas.txt          # Quita escritura al dueño (solo lectura)
echo "nuevo" >> notas.txt    # Error: permiso denegado
3 Compartir un archivo con el grupo 
chmod g+r /home/user/proyecto.txt   # Grupo puede leerlo
chmod g+w /home/user/proyecto.txt   # Grupo puede editarlo
⚠️ Recuerda: para acceder a un directorio necesitas permiso x. Si le das permisos a un archivo dentro de un directorio sin x, los usuarios no podrán atravesar el directorio para llegar al archivo.

Ejercicio final

# Crea un archivo, ajústale permisos progresivamente
touch ejercicio.txt
ls -l ejercicio.txt
chmod u=rw,g=r,o= ejercicio.txt
ls -l ejercicio.txt
chmod u+x ejercicio.txt
chmod g+w,o+r ejercicio.txt
ls -l ejercicio.txt
