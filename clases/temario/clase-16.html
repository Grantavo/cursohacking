Control de atributos de ficheros en Linux - Chattr y Lsattr

Duración: 3:54 · Sección 2: Temario

Los permisos tradicionales (rwx) no son la única capa de seguridad en Linux. Los atributos extendidos con chattr y lsattr permiten controlar propiedades adicionales de los archivos a nivel del sistema de archivos.

Atributos principales

AtributoNombreDescripción
iImmutableEl archivo no se puede modificar, eliminar, renombrar ni enlazar. Ni siquiera root puede hacerlo sin quitar primero este atributo.
aAppend onlySolo se puede añadir contenido al final. Útil para logs.
uUndeletableSe guarda una copia del archivo al eliminarlo (posible recuperación).
cCompressedEl archivo se comprime automáticamente en disco.
sSecure deletionSe sobrescribe con ceros al eliminar (no recuperable).

Comandos básicos

lsattr archivo.txt             # Muestra los atributos de un archivo
lsattr -R /directorio/         # Muestra atributos recursivamente
chattr +i archivo.txt          # Añade el atributo immutable
chattr -i archivo.txt          # Quita el atributo immutable
chattr +a /var/log/syslog      # Añade append-only al log
🔒 Protección crítica: chattr +i /etc/passwd evita que se modifique el archivo de usuarios, incluso si alguien obtiene acceso root. Es una medida de hardening básica.

Ejemplo práctico: proteger /etc/passwd

# Ver atributos actuales
lsattr /etc/passwd
# ----i-------- /etc/passwd   (si ya tiene la i)

# Añadir inmutable
sudo chattr +i /etc/passwd

# Intentar modificar
sudo useradd hacker           # Error: no se puede modificar /etc/passwd

# Para deshacer
sudo chattr -i /etc/passwd
⚠️ El atributo i es tan restrictivo que ni siquiera root puede sobrescribir el archivo. Para modificarlo, debes quitar el atributo primero y luego volver a ponerlo. Úsalo con cuidado en archivos del sistema.

Casos de uso

  • +i en binarios críticos (/bin/ls, /usr/bin/sudo) para evitar modificaciones maliciosas
  • +a en logs (/var/log/auth.log) para que nadie pueda borrar el historial
  • +i en claves SSH (~/.ssh/authorized_keys) para evitar que scripts maliciosos añadan claves
