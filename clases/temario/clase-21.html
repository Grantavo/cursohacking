Uso de bashrc y zshrc

Duración: 8:06 · Sección 2: Temario

Cada vez que abres una terminal, tu shell (bash, zsh, fish, etc.) lee archivos de configuración que definen variables de entorno, alias, funciones y la apariencia del prompt. El archivo más importante es .bashrc (para Bash) o .zshrc (para Zsh). Personalizar estos archivos es el primer paso para construir un entorno de trabajo eficiente.

¿Qué archivos de configuración existen?

ArchivoCuándo se ejecuta
~/.bashrcEn cada nueva terminal interactiva (no login)
~/.bash_profileAl iniciar sesión (login shell)
~/.profileAlternativa a .bash_profile si no existe
~/.bash_logoutAl cerrar la sesión
~/.zshrcEn cada nueva terminal interactiva en Zsh

Variables de entorno: PATH, PS1 y más

Las variables de entorno controlan el comportamiento del shell. Las más importantes:

echo $PATH          # Directorios donde el shell busca ejecutables
echo $PS1           # Formato del prompt
echo $HOME          # Ruta del directorio personal
echo $SHELL         # Qué shell estás usando

Para añadir un directorio al PATH de forma permanente, edita tu ~/.bashrc:

# Añadir ~/.local/bin al PATH
export PATH="$HOME/.local/bin:$PATH"

Alias: atajos que te ahorrarán horas

Un alias es un atajo para comandos largos. Aquí tienes algunos esenciales para hacking:

# En ~/.bashrc
alias ll='ls -la'
alias la='ls -A'
alias l='ls -CF'
alias ..='cd ..'
alias ...='cd ../..'
alias cls='clear'
alias g='grep -i'
alias ports='netstat -tulanp'
💡 Pro tip: Crea un alias alias ip='curl -s ifconfig.me' para conocer tu IP pública al instante. O alias scan='nmap -sV -sC' para escaneos rápidos con nmap.

Personalizar el prompt (PS1)

El prompt puede mostrar información útil: usuario, máquina, directorio actual, hora, etc.

# Prompt básico con colores
PS1='\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$ '

# Prompt con fecha, hora y estado del último comando
PS1='[\[\033[01;32m\]\u\[\033[00m\]@\[\033[01;34m\]\h\[\033[00m\] \t] \w\n\$ '

# Explicación: \u = usuario, \h = hostname, \w = ruta completa, \t = hora

Ejemplo de .bashrc completo

Este es un punto de partida sólido que puedes copiar directamente a tu máquina:

# ~/.bashrc — Configuración personalizada

# --- Variables de entorno ---
export EDITOR=vim
export PATH="$HOME/.local/bin:$PATH"

# --- Prompt personalizado ---
PS1='\[\e[0;32m\]\u\[\e[m\] \[\e[1;34m\]\w\[\e[m\] \[\e[1;33m\]$\[\e[m\] '

# --- Alias ---
alias ll='ls -la'
alias la='ls -A'
alias ..='cd ..'
alias ...='cd ../..'
alias cls='clear'
alias ports='ss -tulanp'

# --- Funciones ---
mkcd() {
    mkdir -p "$1" && cd "$1"
}

extract() {
    if [ -f "$1" ]; then
        case "$1" in
            *.tar.gz)  tar -xzf "$1" ;;
            *.zip)     unzip "$1" ;;
            *.rar)     unrar x "$1" ;;
            *)         echo "Formato no soportado" ;;
        esac
    fi
}

# --- Seguridad: no guardar comandos duplicados en el historial ---
export HISTCONTROL=ignoredups:erasedups
⚠️ Atención: Si editas .bashrc y algo sale mal (el prompt se rompe o no puedes escribir), puedes arreglarlo ejecutando /bin/bash --norc para iniciar Bash sin configuraciones, o editar el archivo directamente con vim ~/.bashrc desde una terminal sin carga.

Después de modificar .bashrc, aplica los cambios con source ~/.bashrc o abre una nueva terminal. Cada usuario en el sistema tiene su propio .bashrc en su $HOME, lo que permite personalizaciones individuales.
