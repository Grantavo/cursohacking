Creación de scripts en Bash

Duración: 9:09 · Sección 2: Temario

Bash no es solo un shell para ejecutar comandos; es un lenguaje de programación completo. Automatizar tareas con scripts es una de las habilidades más valiosas en ciberseguridad: desde escaneos automatizados hasta parsing de logs o despliegue de herramientas. Vamos a construir desde cero.

Estructura básica de un script

Todo script en Bash comienza con el shebang #!/bin/bash, que le indica al sistema qué intérprete usar:

#!/bin/bash
# mi-primer-script.sh — Esto es un comentario

echo "¡Hola, mundo!"

Para ejecutarlo, primero debes darle permisos de ejecución:

chmod +x mi-primer-script.sh
./mi-primer-script.sh

Variables

Las variables almacenan datos. No llevan $ al asignarse, pero sí al usarse:

#!/bin/bash
nombre="Hack4u"
curso="Introducción a Linux"
echo "Bienvenido a $nombre — $curso"

# Variables numéricas
a=5
b=3
suma=$((a + b))
echo "La suma es $suma"

Leer entrada del usuario

#!/bin/bash
echo "¿Cómo te llamas?"
read nombre
echo "Hola, $nombre. Bienvenido al curso."

Condicionales if/else

#!/bin/bash
echo "Introduce tu edad:"
read edad

if [ "$edad" -ge 18 ]; then
    echo "Eres mayor de edad"
else
    echo "Eres menor de edad"
fi

# Operadores: -eq (igual), -ne (no igual), -gt (mayor que)
#             -ge (mayor o igual), -lt (menor que), -le (menor o igual)

# Comprobar si existe un archivo
if [ -f "/etc/passwd" ]; then
    echo "El archivo passwd existe"
fi
💡 Tip: En las condiciones [ ], los espacios alrededor de los corchetes y del operador son obligatorios. [ "$edad" -ge 18 ] funciona; ["$edad"-ge 18] no.

Bucles: for y while

#!/bin/bash
# Bucle for: recorrer una lista
for i in 1 2 3 4 5; do
    echo "Número: $i"
done

# Bucle for con rango (bash 4+)
for i in {1..10}; do
    echo "Iteración $i"
done

# Bucle while: repetir mientras se cumpla una condición
contador=1
while [ $contador -le 5 ]; do
    echo "Contador: $contador"
    contador=$((contador + 1))
done

Argumentos del script

Los argumentos se pasan al script después del nombre y se accede a ellos con $1, $2, $3... $@ contiene todos los argumentos:

#!/bin/bash
# Ejemplo: ./saludo.sh Juan Pérez
echo "Primer argumento: $1"
echo "Segundo argumento: $2"
echo "Todos los argumentos: $@"
echo "Número de argumentos: $#"

Script completo paso a paso: escáner de puertos simple

Vamos a crear un script que escanee puertos abiertos en una IP dada. No es nmap, pero te enseña cómo estructurar un script real:

#!/bin/bash

# escaner.sh — Escáner de puertos básico

if [ -z "$1" ]; then
    echo "Uso: $0 <dirección-ip>"
    exit 1
fi

ip="$1"
echo "Escaneando $ip..."
echo "Puertos abiertos:"

for puerto in 22 80 443 8080 3306 8443; do
    timeout 1 bash -c "echo >/dev/tcp/$ip/$puerto" 2>/dev/null
    if [ $? -eq 0 ]; then
        echo "  [+] Puerto $puerto — ABIERTO"
    fi
done

echo "Escaneo completado."
⚠️ Uso responsable: Este script abre conexiones TCP a los puertos especificados. Solo escanea sistemas sobre los que tengas autorización. Escanear sin permiso puede ser ilegal.

Los scripts en Bash son tu primera línea de automatización. Domínalos antes de pasar a Python o Go, porque Bash es omnipresente en cualquier sistema Linux y no necesitas instalar nada adicional.
