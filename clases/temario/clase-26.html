Uso y configuración de la Kitty

Duración: 10:03 · Sección 2: Temario

Kitty es un emulador de terminal moderno que aprovecha la GPU para renderizar texto de forma ultrarrápida. Es ligero, altamente configurable y se ha convertido en el favorito de muchos hackers y desarrolladores. A diferencia de terminales tradicionales como GNOME Terminal o Konsole, Kitty está pensado para ser manejado casi por completo desde el teclado.

Instalación

# En Debian/Ubuntu
sudo apt install kitty

# En Arch
sudo pacman -S kitty

# En macOS
brew install --cask kitty

Configuración básica: kitty.conf

Kitty se configura desde ~/.config/kitty/kitty.conf. No tiene interfaz gráfica de ajustes; todo se hace editando este archivo:

# ~/.config/kitty/kitty.conf

# Fuente (obligatorio tener la JetBrains Mono instalada)
font_family      JetBrains Mono
bold_font         JetBrains Mono Bold
italic_font       JetBrains Mono Italic
font_size         12

# Opacidad y difuminado del fondo
background_opacity 0.9
dynamic_background_opacity yes

# Esquema de colores oscuro
foreground #dddddd
background #111111
selection_foreground #000000
selection_background #fffacd

# Atajos útiles
map ctrl+shift+c copy_to_clipboard
map ctrl+shift+v paste_from_clipboard
map ctrl+shift+t new_tab
map ctrl+shift+q close_window

Atajos de teclado esenciales

AtajoAcción
Ctrl+Shift+EnterAbrir nueva ventana (panel)
Ctrl+Shift+tNueva pestaña
Ctrl+Shift+TabSiguiente pestaña
Ctrl+Shift+flechasRedimensionar paneles
Ctrl+Shift+lAbrir launcher de Kitty (abreventanas rápido)
Ctrl+Shift+.Buscar en la ventana
💡 Ventaja sobre otras terminales: Kitty renderiza con OpenGL usando la GPU, lo que significa que incluso con mucha salida de texto (logs, nmap -v, grep masivos) no se ralentiza. Además, soporta imágenes inline: puedes ver imágenes directamente en la terminal con kitty +kitten icat imagen.png.

# Ver imágenes inline en Kitty
kitty +kitten icat ruta/a/imagen.png

# Abrir un programa en una ventana flotante
kitty --class floating nvim

# Sesión remota SSH dentro de Kitty
kitty +kitten ssh usuario@host
⚠️ Compatibilidad: Si usas Kitty en un entorno virtualizado, la aceleración GPU puede no estar disponible. En ese caso, funcionará igual, pero sin la ventaja de rendimiento.

Kitty no es solo una terminal bonita; es una herramienta de productividad. Una vez que configures tu kitty.conf con tus colores, fuentes y atajos, notarás que trabajar en la terminal es más fluido y rápido.
