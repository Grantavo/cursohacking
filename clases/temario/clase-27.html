Uso del editor Vim

Duración: 13:20 · Sección 2: Temario

Vim es el editor de texto por excelencia en el mundo Linux. Está presente en prácticamente todos los sistemas Unix y es una herramienta obligatoria para cualquier persona que trabaje en ciberseguridad. Configurar un servidor, editar archivos de configuración remotos por SSH, modificar scripts sobre la marcha — todo se hace con Vim (o su predecesor Vi). La curva de aprendizaje es empinada, pero una vez que lo dominas, editar texto se vuelve intuitivo y rapidísimo.

Los cuatro modos de Vim

Vim no funciona como un editor normal. Tiene modos, y cada modo tiene un propósito:

ModoUsoCómo entrar
NormalNavegar, copiar, pegar, eliminarEsc
InserciónEscribir textoi, a, o
VisualSeleccionar textov, V, Ctrl+v
ComandoGuardar, salir, buscar, reemplazar:

Navegación: h, j, k, l

En modo normal, las teclas h, j, k, l mueven el cursor. Así evitas tener que mover la mano al teclado de flechas:

h       ← izquierda
j       ↓ abajo
k       ↑ arriba
l       → derecha
w       siguiente palabra
b       palabra anterior
0       inicio de línea
$       final de línea
gg      inicio del archivo
G       final del archivo
:42     ir a la línea 42

Guardar y salir

:w      guardar (write)
:q      salir (quit)
:wq     guardar y salir
:q!     salir sin guardar (forzar)
:wq!    guardar y salir forzado
:x      guardar y salir (igual que :wq)
ZZ      guardar y salir (sin escribir :)

Edición básica

dd      eliminar línea completa
yy      copiar línea (yank)
p       pegar debajo del cursor
P       pegar encima del cursor
u       deshacer (undo)
Ctrl+r  rehacer (redo)
x       eliminar carácter bajo el cursor
dw      eliminar palabra
r       reemplazar un carácter

Búsqueda y reemplazo

/patrón     buscar "patrón" hacia adelante
?patrón     buscar "patrón" hacia atrás
n           siguiente coincidencia
N           coincidencia anterior

:s/foo/bar/          reemplazar primera ocurrencia en la línea actual
:s/foo/bar/g         reemplazar todas en la línea actual
:%s/foo/bar/g        reemplazar en todo el archivo
:%s/foo/bar/gc       reemplazar en todo pidiendo confirmación
💡 Tip: Para guardar un archivo que requiere permisos de root y olvidaste usar sudo vim, ejecuta :w !sudo tee % y luego :q!. Vim ejecutará el comando externo tee para escribir el archivo con privilegios elevados.

Configuración .vimrc

El archivo ~/.vimrc personaliza Vim. Aquí tienes una configuración mínima pero potente:

" ~/.vimrc — Configuración esencial

" Números de línea
set number
set relativenumber

" Sintaxis y colores
syntax on
colorscheme desert

" Indentación
set tabstop=4
set shiftwidth=4
set expandtab
set autoindent

" Búsqueda
set incsearch
set hlsearch
set ignorecase
set smartcase

" Ratón habilitado
set mouse=a

" Clipboard del sistema
set clipboard=unnamedplus

" Atajo: ; equivale a : (ahorras un Shift)
nnoremap ; :

" Atajo: jk como Escape en modo inserción
inoremap jk <Esc>
⚠️ Vim no es opcional: En muchos sistemas comprometidos o servidores de producción, Vim es el único editor disponible. No saber usar Vim te dejará sin poder editar archivos en situaciones críticas. Si solo aprendes una herramienta de edición, que sea Vim.

Practica 10-15 minutos al día con vimtutor (el tutorial interactivo que viene con Vim). En una semana tendrás suficiente soltura para editar archivos sin pensar en los comandos.
