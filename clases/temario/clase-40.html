Creamos un descompresor recursivo automático de archivos en Bash

Duración: 23:58 · Sección 2: Temario

En esta clase construimos un script en Bash que detecta automáticamente el formato de compresión de un archivo y lo descomprime, aplicándose recursivamente sobre cualquier resultado extraído. Es una herramienta muy útil en CTFs y análisis de malware donde los archivos vienen anidados en múltiples capas.

Detección del tipo de archivo con file

El comando file examina la cabecera mágica de un archivo para determinar su tipo real, independientemente de la extensión.

# Detectar tipo de archivo
file datos.bin
# Salida: datos.bin: gzip compressed data, was "backup.tar"

# En el script usaremos file --mime-type para obtener el tipo MIME
file --mime-type -b archivo.zip
# Salida: application/zip
📖 ¿Cómo funciona file? Lee los primeros bytes (magic numbers) del archivo. Por ejemplo, un ZIP empieza con PK\x03\x04, un gzip con \x1f\x8b, un bzip2 con BZh. Esto permite identificar el formato aunque la extensión sea incorrecta.

Estructura del script: case/esac

Usamos case para seleccionar el comando de descompresión según el tipo MIME detectado.

#!/bin/bash

descomprimir() {
    local archivo="$1"
    local tipo
    tipo=$(file --mime-type -b "$archivo")

    case "$tipo" in
        application/gzip)
            mv "$archivo" "${archivo}.gz"
            gunzip "${archivo}.gz"
            ;;
        application/x-bzip2)
            bunzip2 "$archivo"
            ;;
        application/x-xz)
            unxz "$archivo"
            ;;
        application/zip)
            unzip -o "$archivo"
            ;;
        application/x-tar)
            tar xf "$archivo"
            ;;
        *)
            echo "Formato no soportado: $tipo"
            return 1
            ;;
    esac
}

Bucle recursivo con while

El corazón del script: mientras haya archivos comprimidos, los descomprime y procesa los resultados.

#!/bin/bash

procesar_archivos() {
    while true; do
        local comprimido
        comprimido=$(find . -type f 2>/dev/null | while read -r f; do
            case $(file --mime-type -b "$f") in
                application/gzip|application/x-bzip2|application/x-xz|application/zip|application/x-tar)
                    echo "$f"
                    break
                    ;;
            esac
        done)

        [ -z "$comprimido" ] && break

        echo "[+] Descomprimiendo: $comprimido"
        (cd "$(dirname "$comprimido")" && descomprimir "$(basename "$comprimido")")
    done
    echo "[*] No hay más archivos comprimidos."
}

procesar_archivos
⚠️ Precaución: Un descompresor recursivo puede ser peligroso si el archivo es un zip bomb — un archivo diseñado para expandirse enormemente. Siempre usa unzip -l para inspeccionar el contenido antes de extraer archivos sospechosos.

El script completo incluye además manejo de errores, colores para la salida, soporte para 7z y rar, y opción de límite de profundidad recursiva. Puedes descargarlo desde el repositorio del curso.
