Creando nuestros propios escáneres en Bash

Duración: 21:29 · Sección 2: Temario

En esta clase construimos un escáner de puertos en Bash puro usando el pseudo-sistema de archivos /dev/tcp. Es una técnica que funciona en cualquier sistema con Bash y no requiere herramientas externas como nmap o netcat.

El pseudo-fichero /dev/tcp

Bash permite abrir conexiones TCP a través de /dev/tcp/IP/PORT. Si la conexión tiene éxito, el puerto está abierto.

# Probar un puerto específico
bash -c "echo > /dev/tcp/192.168.1.1/80" 2>/dev/null && echo "Puerto 80 abierto" || echo "Puerto 80 cerrado"

# Con timeout para evitar esperas largas
timeout 1 bash -c "echo > /dev/tcp/192.168.1.1/22" 2>/dev/null && echo "Abierto" || echo "Cerrado"
🧠 ¿Cómo funciona? Bash interpreta /dev/tcp/host/puerto como una redirección especial. Si la conexión TCP se establece, la redirección tiene éxito. No requiere permisos especiales ni netcat. No funciona en shells como sh o dash; necesita Bash compilado con --enable-net-redirections.

Escáner con bucle for

Iteramos sobre un rango de puertos con un bucle for y probamos cada uno:

#!/bin/bash
ip="$1"
echo "Escaneando $ip..."
for port in {1..1024}; do
  timeout 1 bash -c "echo > /dev/tcp/$ip/$port" 2>/dev/null && echo "Puerto $port -> ABIERTO"
done
echo "Escaneo completado."

Mejoras: colores y paralelización

El escáner secuencial es lento. Podemos paralelizar con & (background) y añadir colores para mejor legibilidad:

#!/bin/bash
ip="$1"
verde="\e[32m"
rojo="\e[31m"
reset="\e[0m"

echo -e "${verde}[+] Escaneando $ip...${reset}"

for port in {1..5000}; do
  (
    timeout 1 bash -c "echo > /dev/tcp/$ip/$port" 2>/dev/null && \
    echo -e "${verde}[ABIERTO]${reset} $port"
  ) &
done

wait
echo -e "${verde}[+] Escaneo completado.${reset}"
⚠️ Control de procesos: Lanzar todos los puertos en paralelo (5000 procesos) puede saturar el sistema. Es mejor usar un control de concurrencia con jobs y wait limitando a, por ejemplo, 100 procesos simultáneos: 
max_jobs=100
for port in {1..5000}; do
  ( ... ) &
  while [ $(jobs -r | wc -l) -ge $max_jobs ]; do
    sleep 0.1
  done
done

El proyecto completo incluye detección de servicios (banner grabbing), exportación a CSV, y modo sigiloso con retardos aleatorios. Disponible en el repositorio del curso.
