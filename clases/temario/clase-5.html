Comandos básicos de Linux [1-2]

Duración: 10:46 · Sección 2: Temario

En esta clase aprenderás los comandos esenciales para moverte y manipular archivos en Linux. Dominar estos comandos es el primer paso para sentirte cómodo en la terminal.

Estructura de sintaxis

Todo comando en Linux sigue esta estructura general:

comando [opciones] [argumentos]

Las opciones modifican el comportamiento del comando (suelen ir precedidas de - o --), y los argumentos son los objetivos sobre los que actúa (archivos, directorios, etc.).

Navegación: pwd, ls, cd

pwd (print working directory) muestra la ruta absoluta del directorio actual:

pwd
/home/usuario

ls lista el contenido de un directorio:

ls          # Listado simple
ls -l       # Listado detallado (permisos, dueño, tamaño, fecha)
ls -la      # Listado detallado incluyendo archivos ocultos
ls -lh      # Tamaños en formato legible (KB, MB)

cd cambia de directorio:

cd /tmp            # Ir a /tmp
cd ~               # Ir al home del usuario
cd ..              # Subir un nivel
cd -               # Volver al directorio anterior
💡 Tip: Usa la tecla Tab para autocompletar rutas y nombres de archivos. Si hay múltiples coincidencias, pulsa Tab dos veces para verlas.

Creación de archivos y directorios

mkdir crea directorios. touch crea archivos vacíos o actualiza su fecha de modificación:

mkdir proyectos                    # Crea un directorio
mkdir -p proyectos/web/css         # Crea toda la estructura de subdirectorios
touch notas.txt                    # Crea un archivo vacío
touch -t 202401011200 notas.txt    # Cambia la fecha a 1 Ene 2024 12:00

Eliminación: rm y rmdir

rm archivo.txt                     # Elimina un archivo
rm -r directorio/                  # Elimina un directorio y su contenido
rm -rf directorio/                 # Elimina sin pedir confirmación (¡cuidado!)
rmdir directorio_vacio/            # Solo elimina directorios vacíos
⚠️ Precaución: rm -rf no pide confirmación. Un rm -rf / borraría todo el sistema. Siempre revisa antes de ejecutar.

Copiar y mover: cp y mv

cp origen.txt destino.txt          # Copia un archivo
cp -r carpeta/ backup/             # Copia un directorio recursivamente
mv viejo.txt nuevo.txt             # Renombra un archivo
mv archivo.txt ~/Documentos/       # Mueve un archivo a otro directorio

Visualización: cat, echo

cat archivo.txt                    # Muestra el contenido completo
cat -n archivo.txt                 # Muestra con números de línea
echo "Hola mundo"                  # Imprime texto en la terminal
echo "texto" > archivo.txt         # Escribe "texto" en archivo (sobrescribe)
echo "más texto" >> archivo.txt    # Añade "más texto" al final

Ayuda integrada: man

El manual es tu mejor amigo. Cualquier comando se puede consultar con:

man ls           # Muestra el manual de ls
man -k permisos  # Busca páginas que contengan "permisos"
man 5 crontab    # Muestra la sección 5 del manual (formatos)
📖 Dentro de man navega con las flechas, busca con / y sal con q. Las páginas están organizadas en secciones: 1=comandos, 2=llamadas al sistema, 3=liberías, 5=formatos, 8=administración.

Ejercicio práctico

Crea esta estructura de directorios y archivos en tu terminal:

mkdir -p ~/practica/{documentos,scripts,backup}
cd ~/practica
touch notas.txt scripts/hola.sh
echo '#!/bin/bash' > scripts/hola.sh
echo 'echo "Hola desde la terminal"' >> scripts/hola.sh
cp scripts/hola.sh backup/
ls -laR
