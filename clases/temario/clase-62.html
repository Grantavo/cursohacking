Cuestionario de conceptos y comandos en Linux

Duración: — · Sección 2: Temario

Este cuestionario final cubre todos los temas de la sección Temario. Responde cada pregunta para comprobar tu dominio de Linux.

Cuestionario final

1. ¿Qué comando muestra el directorio actual de trabajo?

  • a) ls
  • b) pwd
  • c) cd
  • d) dir

2. ¿Qué permisos representa 755 en octal?

  • a) rwxr-xr-x
  • b) rwxrwxrwx
  • c) rwxr--r--
  • d) r-xr-xr-x

3. ¿Qué flag de find permite ejecutar un comando por cada archivo encontrado?

  • a) -exec
  • b) -ok
  • c) -delete
  • d) -print

4. ¿Qué hace el comando grep -r "texto" .?

  • a) Busca "texto" recursivamente en todos los archivos del directorio actual
  • b) Reemplaza "texto" en el directorio actual
  • c) Muestra las líneas que no contienen "texto"
  • d) Cuenta las ocurrencias de "texto"

5. ¿Qué comando de sed reemplaza la primera ocurrencia de "foo" por "bar" en cada línea?

  • a) sed 's/foo/bar/'
  • b) sed 's/foo/bar/g'
  • c) sed 's/bar/foo/'
  • d) sed 's/foo/bar/2'

6. En awk, ¿qué variable representa el primer campo de una línea?

  • a) $0
  • b) $1
  • c) NR
  • d) NF

7. ¿Qué expresión cron ejecuta un comando cada hora en punto?

  • a) 0 * * * *
  • b) * 0 * * *
  • c) 0 0 * * *
  • d) * * * * *

8. ¿Qué puerto utiliza SSH por defecto?

  • a) 22
  • b) 80
  • c) 443
  • d) 3306

9. ¿Qué comando de netcat escucha conexiones entrantes en el puerto 4444?

  • a) nc -lvnp 4444
  • b) nc 127.0.0.1 4444
  • c) nc -u 4444
  • d) nc -z 4444

10. ¿Qué comando se usa para iniciar una sesión SSH con un usuario específico?

  • a) ssh usuario@ip
  • b) ssh ip usuario
  • c) ssh -u usuario ip
  • d) ssh login usuario ip

11. En Git, ¿qué comando descarga un repositorio remoto por primera vez?

  • a) git clone
  • b) git pull
  • c) git fetch
  • d) git init

12. ¿Qué flag de chmod permite cambiar permisos de forma recursiva?

  • a) -R
  • b) -r
  • c) -v
  • d) -f

13. ¿Qué variable de Bash contiene el número de argumentos pasados a un script?

  • a) $#
  • b) $@
  • c) $*
  • d) $0

14. ¿Qué herramienta de fuerza bruta es más conocida para auditorías SSH?

  • a) Hydra
  • b) Nmap
  • c) Wireshark
  • d) John the Ripper

15. ¿Qué carácter se usa en Bash para escapar un comando dentro de otro mediante subshell?

  • a) $(...)
  • b) {...}
  • c) [...]
  • d) (...)

16. ¿Qué hace el comando git stash pop?

  • a) Recupera el último stash guardado y lo elimina de la lista
  • b) Elimina todos los stashes
  • c) Muestra la lista de stashes
  • d) Crea un stash nuevo
